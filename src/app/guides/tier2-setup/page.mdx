# Tier 2 — Dedicated Node Setup

*This guide will be available when Tier 2 hardware specifications are finalized.*

Tier 2 nodes are purpose-built for 24/7 operation with enhanced data quality. The hardware package will include:

- **Single-board computer** (Raspberry Pi 5 or equivalent)
- **RTL-SDR V4** USB dongle
- **GPS-disciplined oscillator (GPSDO)** for nanosecond-accurate timing
- **Outdoor antenna** with weatherproof enclosure
- **Pre-configured software image** — flash and go

Tier 2 nodes earn higher rewards than Tier 1 because GPS-disciplined timing enables more precise observations that anchor the network's accuracy.
