# Tier 3 — Coherent Array Setup

*This guide will be available when Tier 3 hardware specifications are finalized.*

Tier 3 nodes use multi-channel coherent receivers such as the KrakenSDR to enable advanced positioning techniques beyond simple TDOA.

## What to Expect

- **Hardware:** KrakenSDR or equivalent 5-channel coherent SDR (~$400–600)
- **Antenna:** 5-element uniform circular array for Angle-of-Arrival estimation
- **Capabilities:** Simultaneous TDOA + AoA measurements, enabling single-node position fixes
- **Timing:** GPS-disciplined oscillator with nanosecond-class synchronization
- **Rewards:** Highest base points (15 per epoch) with the lowest saturation decay

## Requirements

- Outdoor antenna placement with clear sky view for GPS
- Stable internet connection (wired preferred)
- Dedicated Linux or Windows host running 24/7
- Sufficient USB bandwidth for 5-channel SDR
